# abieri1023.github.io

个人生活工具集合小程序，一个轻量级的日常工具集，包含排班查询、地图足迹和余额记录三大功能。

## 项目简介

本项目是一个纯前端的生活工具集合，无需后端服务器即可运行。通过浏览器即可快速访问和使用所有功能，数据存储在本地 localStorage 中，保护用户隐私。

## 整体架构

项目采用纯 HTML + CSS + JavaScript 架构，无任何外部框架依赖：

```
├── index.html      # 排班小程序首页
├── address.html    # 到此一游 - 旅行足迹地图
├── balance.html    # 余额记录
├── home.ico        # 网站图标
└── sw.js          # Service Worker（屏蔽百度统计）
```

### 页面功能说明

#### 1. 排班小程序（index.html）
- **排班日历**：以5天为周期循环显示班次（白班1、白班2、夜班1、夜班2、休息）
- **近期班次**：快速查看昨天、今天、明天、后天的排班情况
- **日期计算器**：计算距离目标日期的天数差
- **打卡功能**：提供微信/支付宝快捷跳转入口
- **反馈按钮**：「好用」和「难用」趣味反馈

#### 2. 到此一游（address.html）
- **百度地图展示**：使用百度地图 API 在网页中展示旅行足迹
- **城市切换**：支持快速切换查看多个城市的足迹标记
- **覆盖城市**：江油、绵阳、成都、自贡、宜宾、乐山、昆明、上海、杭州、苏州、嘉兴、嘉善、无锡、昆山

#### 3. 余额记录（balance.html）
- **多账户管理**：支持银行卡、支付宝、微信、其他四种分类
- **记录添加**：输入金额和备注即可添加记录
- **数据统计**：自动计算当日合计和总余额
- **本地存储**：所有数据存储在浏览器 localStorage 中

## 技术特性

- **响应式设计**：适配手机和电脑端浏览
- **无后端依赖**：纯前端实现，部署简单
- **本地数据存储**：使用 localStorage 保存用户数据
- **轻量级**：单文件实现，功能完整
- **百度地图 API**：用于地理位置展示

## 安装教程

本项目为零依赖纯前端项目，部署极为简便：

### 方式一：直接部署到 GitHub Pages

1. 将所有文件上传到 GitHub 仓库
2. 在仓库 Settings → Pages 中启用 Pages 功能
3. 选择 `main` 分支和 `/ (root)` 目录
4. 访问 `https://[username].github.io/[repository]` 即可

### 方式二：本地预览

1. 下载或克隆本仓库
2. 直接在浏览器中打开 `index.html` 文件即可使用

> **注意**：部分浏览器（如 Chrome）可能因安全策略限制本地文件访问地图等功能，建议使用本地服务器或直接部署到 GitHub Pages。

## 使用说明

### 排班小程序
1. 打开首页自动显示当月排班日历
2. 点击「← 上个月」或「下个月 →」切换月份查看
3. 页面顶部显示近四天班次安排
4. 在日期计算器中选择目标日期，点击「计算」查看天数差
5. 点击「好用」或「难用」可查看彩蛋效果

### 到此一游
1. 点击城市按钮可切换显示不同城市的地图位置
2. 支持地图缩放和平移操作
3. 如遇卡顿，请刷新页面重新加载

### 余额记录
1. 选择账户类型（银行卡/支付宝/微信/其他）
2. 输入金额，可选填写备注
3. 点击「添加记录」保存
4. 点击「清空当日记录」可清除当天所有记录
5. 数据自动保存在浏览器本地

## 其他说明

### 浏览器兼容性
- 推荐使用 Chrome、Firefox、Safari、Edge 等现代浏览器
- 部分功能依赖 JavaScript，请确保浏览器已启用 JavaScript

### 数据存储
- 所有用户数据存储在浏览器 localStorage 中
- 清除浏览器数据将导致记录丢失，请注意备份重要数据

### 第三方依赖
- 百度地图 JavaScript API（用于地图展示）
- 无其他外部依赖

### 部署建议
- 推荐部署到 GitHub Pages 免费托管
- 也可部署到任意静态文件托管服务（阿里云OSS、七牛云等）
